## The five-line summary

- Shorts ads are tiny; the channel is the **funnel**, not the business.
- Near-term target: **500 subs** (fan-funding tier + Shopping stickers), then full YPP via the
  **long-form watch-hours door** — which is also where finance money actually lives ($10–25 RPM).
- Build the **owned audience** (newsletter: $30–100/subscriber/yr) starting now.
- Our moat: **production cost ≈ $0** where competitors pay $500–2k/mo — so clone channels when proven.
- Exit optionality at 18–24× monthly profit; the codebase itself is a saleable asset.

## How this folder stays alive

The weekly **ShortsLearn** run reviews the MASTER_PLAN KPIs (subs delta, views/90d run-rate, AVP,
platform thresholds) and appends horizon status to `learnings.md`; milestone unlocks get flagged in
`DECISIONS_FOR_OWNER.md`. Update the plan when experiments (E1–E3) or milestones change the picture —
it's a living document, not a shelf document.
