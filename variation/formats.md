# Content-format library (rotate — vary the NARRATIVE SHAPE so videos aren't templated)

The producer picks ONE format per video that wasn't used in the last ~5 videos (`variation_ledger.md`).
Each format changes the script structure AND suggests visual/pacing differences. Keep every hard rule
(question/hook/what-if opener, validate-before-render, hand-coded visuals, unique-value layer).

| id | Format | Structure (script shape) | Pairs well with |
|---|---|---|---|
| F1 | **Explainer** | hook → setup → mechanism → ONE hero reveal → payoff. "how/why X really works." | skins Cinematic/Blueprint; titles T15,T24 |
| F2 | **Myth-bust** | state the common belief → "actually…" hard reversal → proof → what to do instead. | skins Bright/Blueprint; titles T05,T06,T18 |
| F3 | **Listicle** | hook → "here are N…" → beat per item (N=3) → strongest last → recap button. | skins Data/Bright; titles T13,T08 |
| F4 | **Story** | a "you/your friend" character hits an escalating problem → consequence → the fix (à la Gohar Khan). | skins Paper-Cutout/Cinematic; titles T22,T12,T19 |
| F5 | **Comparison** | A vs B side-by-side → which wins on the metric that matters → the takeaway. | skins Data/Blueprint; titles T17 |
| F6 | **Test-it-yourself** | hook → "do this right now" participation → what you'll see → why it matters. | skins Cinematic/Neon; titles T25,T10 |
| F7 | **Case-study / number** | one shocking real number → dissect where it comes from → what it means for you. | skins Data-Room/Neon; titles T11,T16,T21 |
| F8 | **What-most-people-miss** | the thing everyone overlooks → why it's counterintuitive → the payoff of knowing it. | skins Blueprint/Cinematic; titles T03,T04,T09 |

**Rules:** don't repeat a format id within 5 videos; the format DICTATES the script skeleton (a listicle is
3 ranked beats, a comparison is two-column, a story has a character) so consecutive videos genuinely differ
in shape, not just topic. Choose the format that best fits the topic + the chosen title-shape + skin.
